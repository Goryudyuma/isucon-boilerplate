# isucon-boilerplate

## usage

### initialize

```bash
cd /
git init
git remote add origin git@github.com:bgpat/isucon-boilerplate.git
git fetch origin master
git reset --hard FETCH_HEAD
make
```

### edit config file

```bash
vim /etc/nginx/nginx.conf
```

### track source files by git

```bash
git add -f *.go
```
